Durata: 1 ora e 45 minuti

Parte I

  1. Scrivere un metodo statico maxPositiviConsecutivi (possibilmente ricorsivo) che, data una LinkedList (o un array) di Integer, restituisce la lunghezza della più lunga sottosequenza contenente solo valori positivi. Ad esempio, data la sequenza [3, 4, -2, 4, 7, 3, 6, -7, 2, -1, -5, 3, 4], maxPositiviConsecutivi dovrebbe restituire 4.
  2. Il cammino destro di un albero è la sequenza di nodi che si incontrano partendo dalla radice e andando sempre sul figlio destro. Scrivere un metodo statico sommaCamminoDestro che, dato albero binario i cui nodi contengono ciscuno un Integer, restituisce la somma dei valori nei nodi sul cammino destro.

Parte II

Domanda 1

Dire cosa stampa il seguente programma:

public class Mistero {
    public static int mistero(int x){
        if (x==0) return 0;
        else if (x>0) return -1+mistero(x-1);
        else return 1+mistero(x+1);
    }
    public static void main(String[] args){
        System.out.println(mistero(5));
        System.out.println(mistero(-5));
    }
}

Quale funzione matematica f(x) è calcolata dal metodo mistero? Motivare le risposte (N.B. risposte non motivate saranno considerate nulle).
