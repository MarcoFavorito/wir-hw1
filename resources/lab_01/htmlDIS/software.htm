SOFTWARE

It's necessary to install Microsoft Active Sync on your computer before effecting any operation. Active Sync allows to effect the synchronization between your computer and the pda.

Such software is available to the web page: http://www.microsoft.com/downloads/details.aspx

Our application has been thought for two types of consumers:

Programmer Version
This version is available to the informatic programmer's for building new applications on the middleware MOBIDIS.
 
User Version
This version allow to the normal user to use the middleware on your mobile device, with the application Manet Chat already build.
