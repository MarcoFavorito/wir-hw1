Regole per la prova di conoscenza del linguaggio

La prova di conoscenza del linguaggio Java è costituita da un test a risposta multipla. Anche questa prova teorica verrà corretta mediante una procedura di correzione automatica. Le risposte vanno inserite in un file di testo risposte.txt, da scrivere con un editor di testo, la cui struttura deve essere la seguente:

NumeroDomanda Risposta

Ad esempio, supponiamo che si voglia rispondere A) alla prima domanda, D) alla seconda, B) alla terza, si voglia omettere la risposta alla quarta domanda e si voglia rispondere A) alla quinta; il file delle risposte è il seguente: risposte.txt

1 A
2 D
3 B
5 A
Qualunque variazione dal formato sopra indicato può rendere impossibile la verifica del risultato, ed in questo caso non garantisce la correzione dell'elaborato
Per evitare problemi di formato si suggerisce di scrivere il file di risposte usando l'editor Notepad. correttezza della verifica.
