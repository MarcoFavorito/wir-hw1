If you are interested in any of these courses please contact Roberto Baldoni.

Wireless networks

This course illustrates the typologies and carachteristics of the main three wirless networks' classes: cellular networks, wireless LANs, mobile ad-hoc networks. The course will be expecially focused on the protocols that constitute the basis of these networks.

CORBA (basic)

Topics: basic notions on distributed systems, object-based distributed computation systems, services for distributed applications, Common Object Request Broker Architecture (Object Request Broker, Object Adapters), examples of simple CORBA applications (Interface Definition Language, stub, skeleton e naming service).

Laboratory: included

Duration: three days

CORBA (advanced)

Topics: dynamic invocation interface and dynamic skeleton interface (DII e DSI), server-side CORBA programming (Portable Object Adapter -POA-, server configuration, implementation strategies), advanced CORBA services (transaction, event, notification services), portable interceptors.

Laboratory: included

Duration: three days

Fault Tolerant CORBA

Topics: basic notions on higly reliable systems and software replication, techniques and strategies for replication, extensions to standard CORBA introduced with Fault Tolerant CORBA (object group, object group reference and components for replication management), server replication, replication management. Examples of simple, high-available, FT-CORBA compliant servers.

Laboratory: included (based on IRL platform)

Duration: five days

Group Communication Design and Development

Topics: message-passing based distributed systems, guarantees on reliability and message ordering (FIFO, causal, total order multicast), group communications and group communication services (membership and state transfer services), architecture of group communication toolkits and their performances. Distributed applications implementation via group communication toolkits: how to realize an high-available replicated server.

Laboratory: included

Duration: three to five days
