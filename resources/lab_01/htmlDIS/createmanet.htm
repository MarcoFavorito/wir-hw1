CREATE MANET NETWORK


The creation of Manet network is possible if you have a wireless internal or external card.

How to star to create ad hoc network on laptop.

The creation of a network ad hoc can be made from whatever computers. If the computers don't have the wireless internal card, all it takes is using an any card Wi-Fi.
 
1. The first step is to install the drivers of wireless card.
2. On panel control, you choose network connection, and then wireless connection, with right key you choose activated.
3. You choose with right key property.
4. If you choose wirelless network you will see all network that you can used for connection.
For greater details him face reference to the appendix "Hardware and Software": click here .
To be able to create a network ad hoc needs to perform the followings footsteps:
1. to specify the IP address, to identify the machine inside a network ad hoc (it is comfortable to use the class of IP address "C '', that it possesses 21 bit of " networkID '' and 8 bit of " hostID ''; such class uses addresses beginning from 192.0.0.0 to reach 223.255.255.255);
2. to choose wireless network, to press the key " add", will appear screened in
which a SSID is required, that identifies the name of the network ad hoc
what he/she is wanted to create.

How to star to create ad hoc network on PDA.

The creation of a Manet network is related to the device followings:
1. IPAQ 5550, with operating system windows CE
2. IPAQ serie 5540 with operating system windows CE (the difference with the device IPAQ 5550 concern the communicative channel, which must be plans to the beginning of the creation of the network ad hoc).
The footsteps that are had to complete to create a network ad hoc among different device PDA are very similar to those that are done for the laptop, particularly:
1. to select start, and to choose impostazioni,
2. to select connections
3. to select advances and to choose card of network
4. once selected the card of network a textBox will appear where are shown all the wireless networks
5. to select the textBox to choose the card of network (in our case the PDAs IPAQ series 5550 and 5540 have one card of network wireless WLAN IPAQ)
6. as last thing to plan the address IP of the PDA, this it will be the unique inside the network.
The procedure of formulation of the address IP is similar tolaptop device. Also in this case it is useful to use addresses IP of the class C.
Once performed such footsteps him it is ready to use a connection ad hoc.
