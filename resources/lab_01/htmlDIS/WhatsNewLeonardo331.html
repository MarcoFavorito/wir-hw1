What's new in Leonardo 3.3.1 ?

  • Hierarchical programs menu
  • New horizontal reduced process control palette (removed vertical one)
  • Faster C compiler
  • Improved memory fault checkings in C library functions where a pointer is passed as argument (for program debugging purposes)
  • Updated on-line reference
  • Automatic destruction of terminated processes in non-debug mode
  • 'Undeallocated blocks' warnings raised only in debug mode
  • Added pane descriptions in the message bar for the project settings window
  • Changes to the program repository:
    • Added a new visit-based tree drawing algorithm to the 'Graph Drawing' folder
    • Added a rudimentary benchmark program to the 'Utilities' folder
    • Added non-reversible Tetris to the 'Games' folder
    • Minor revisions of programs: MinMax, Dijkstra, Huffman, Ball
