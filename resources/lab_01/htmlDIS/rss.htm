RSS feed

Lo standard RSS (Really Simple Syndication) consente di distribuire informazioni attraverso dei 'canali': sottoscrivendo questi canali si ottiene, di fatto, un file XML che è possibile riutilizzare sulle proprie pagine web attraverso, ad esempio, un XSLT.

Altri modi di fruire di un feed RSS sono tramite l'utilizzo di un client apposito, come Abilon e Radio Userland per Windows oppure NewsMac per Mac OsX, o anche installando un'estensione nel proprio browser come Sage per Mozilla FireFox oppure RSS viewer per Konfabulator.
