Download JGISEdit  
   
 

 

JGISEdit is a multiplatform interactive graphical enviroment for creating and editing the vectorial maps used by the RoboCup-Rescue Simulator.

You can download it from here.

Read this page for installation instructions and usage reference.

A snapshot

Minimum configuration

Pentium MMX 166 MHz (Recommended Pentium III 1'200 MHz or higher)
32 megabytes of RAM
2 megabytes of free hard disk space

Installation

JGISEdit requires the Java Virtual Machine v. 1.3.1 (or higher) installed.
If the JVM is correctly installed on your PC, you only need to type java -jar JGISEdit.jar at the command shell of your operating system for running JGISEdit.

For Unix/Linux users: you need RW rights in the application directory.

Main features

CREATING a new RoboCup-Rescue map

- Click on the 'New' button on the toolbar or select the 'New...' item from the 'File' menu.
- Specify the map extension
- If you want, you can specify the full path of an image file (.GIF format) for using as a background image for your map. It's useful if you have to vectorize a raster map; the selected image will be resized to fit the whole map.

OPENING an existent map

- Click on the 'Open' button on the toolbar or select the 'Open...' item from the 'File' menu.
- Use the browser window to find the .bin files of the map in your file system.
- Select any .bin file.

SAVING a map

- Click on the 'Save' button or select the 'Save...' item from the 'File' menu..
- Use the browser window to find the destination directory.
- Type the name of a .bin file (building.bin/ road/node) and click SAVE.

QUITTING

- Select the 'Exit' item from the 'File' menu..

FINDING an object on the map

Menu 'Edit' -> 'Find object' : this function allow to find an object by knowing its ID.
The object with the specified ID, if exists, will be selected.

SELECTING one or more map objects

- Just click on the object you want to select.
- Holding down the CTRL key while clicking allows to select more than one object.

Viewing/editing the selected object(s) PROPERTIES.

- Click on the 'Properties button or select the 'Properties...' item in the 'View' menu.
- Double-click on the cell property in properties window or click the EDIT button.

Map editing features

DELETING one or more map objects

- Select the object(s).
- Hit the 'Delete' key.

Drawing a BUILDING

- Click on the 'Add Building' button or select the 'Add Building...' item in the 'Tools' menu.
- Click on the map for specifying each building vertex.
- Right-Click to create the building.
- Hit ESC to stop using this tool.

Drawing a NODE

- Click on the 'Add Node' button or select the 'Add Node...' item in the 'Tools' menu.
- Click on the map to specify the node position.
- Hit ESC to stop using this tool.

Drawing a ROAD

- Select a node to be used as the "head" node for the road.
- Click the 'Add Road' button or select the 'Add Road...' item in the 'Tools' menu.
- Click on a "tail" node or on another road.
- Hit ESC to stop using this tool.


alternatively :

- Select two or more nodes in sequence (holding down the CTRL key while clicking)
- Click the 'Connect nodes' button or select the 'Connect nodes...' item in the 'Tools' menu to connect the nodes by road segments.

Splitting a road with an in-between node.

- Select the road.
- Click the 'Split Road' button or select the 'Split Road...' item in the 'Tools' menu.
- Click on the map to specify the new node position.

CONNECTING a building to a road

- Select the building.
- Click the 'Connect building' button or select the 'Connect building...' item in the 'Tools' menu.
- Click on the road you want connect to.
- Hit ESC to stop using this tool.

DISCONNECTING a building from a road

- Select the building.
- Click the 'Disconnect building' button or select the 'Disconnect building...' item in the 'Tools' menu.
- Click on the connection node on the road.

Drawing a SHINDO/GAL POLYDATA polygon

- Click the 'Draw ShindoPolydata' / 'GalPolydata' button or select the corresponding item in the 'Tools' menu.
- Click on the map to specify each polygon vertex.
- Right-Click to create the polygon.

 

Map-checking feature

The 'Check world' function in the 'Edit' menu allow to check the map for various inconsistencies:

- "Unconnected" Nodes.
- Building with a bad-formed polygon plant or without access points.
- Roads with the head node same as tail node.
- Not-connected road-graph.

 

Agents editing features

ADDING a new agent/firepoint to the map

- Click the appropriate 'Add agent' button or select the 'Add agent' item in the 'Agents' menu.
For mobile agents:
- click on a building/node/road to set the agent position.
For motionless agents:
- click on a building.

MOVING an agent on the map to a new location.

- Select the agent.
- Click the 'Move object' button or select the 'Move object' item in the 'Tools' menu.
- Click on the new position on the map.

Import/Export features (MS Windows platform only)


EXPORTING a map to a MS Access database.

- Click the 'Export to MS Access' item in the 'File' menu..
- Choose the destination folder.
- Type a (.MDB) filename and hit the SAVE button.

IMPORTING a map from a MS Access database.

- Click the 'Import from MS Access' item in the 'File' menu..
- Select a .MDB file (it must be a file created by the export feature of JGISEdit)
