Installazione GCC/FLEX/BISON

Il software necessario per svolgere le esercitazioni e realizzare il progetto in C e' il seguente:
  • GCC, compilatore per il linguaggio C
  • programma FLEX (LEX)
  • programma BISON (YACC)
Questo software e' disponibile (public-domain) su tutte le piattaforme hw/sw. In particolare:
  • LINUX: chi ha un PC con Linux non dovrebbe avere problemi, in quanto tutto il software necessario (compilatore GCC, FLEX e BISON) e' gia' compreso nelle normali distribuzioni di Linux. Tuttavia, qualche distribuzione non comprende FLEX e BISON: il software per l'installazione di questi tool può essere scaricato dai seguenti link:
    • FLEX
    • BISON
  • WINDOWS: per chi ha un PC con Windows (3.1, 95/98, NT), e' possibile installare l'ambiente di programmazione DJGPP, che comprende GCC, FLEX e BISON:
    • ambiente di programmazione DJGPP (per Windows 95 e Windows 3.1)
    • istruzioni per l'installazione di DJGPP
    • istruzioni per l'uso di FLEX e BISON con DJGPP
    • alcuni consigli sulla gestione della memoria in GCC
    • ambiente di sviluppo GNU-Win32
