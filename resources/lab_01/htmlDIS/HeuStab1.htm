An augmentation algorithm for the maximum weighted stable set problem

  • Authors : Carlo Mannino and Egidio Stefanutti
  • Abstract : Edge projection is a specialization of Lovasz and Plummer's clique reduction when restricted to edges. A concept of augmenting sequences of edge-projections is defined w.r.t. a given stable set. It is then proved the equivalence between the optimality of S and the existence of an augmenting sequence w.r.t. S. This result is then exploited to develop a new tabu-search heuristic for the Maximum Stable Set Problem (weighted and unweighted). The main component of this heuristic is an augmentation algorithm which tries to find augmenting sequences of edge projection in order to improve the weight of a given stable set $S$. The resulting code proved to be competitive with the best codes presented in the literature.
