La prova dura un'ora e tre quarti.

Riempire il file studenteid.txt con i propri dati.

Vengono dati: due programmi di prova, un file di prova, e il testo del compito in formato html. Lo studente deve realizzare due classi che vengono usate dai due programmi di prova. Le soluzioni devono trovarsi nella stessa cartella Esame dove si trovano i programmi di prova pena l'annullamento dell'esame.

Prima parte

Una pizzeria che fa consegne a domicilio vuole informatizzare la gestione delle consegne. Per questo scopo, vanno realizzate due classi. Le prima classe si chiama Pizza, e ogni suo oggetto rappresenta un specifica pizza che è stata ordinata. Le componenti di questo oggetto sono:

tipo
una stringa che indica il tipo di pizza
indirizzo
l'indirizzo a cui va consegnata la pizza
prezzo
il prezzo della pizza in centesimi (un intero)
consegnato
questa componente boolean vale true oppure false a seconda se la pizza è stata già consegnata oppure no

Definire la classe incapsulata Pizza i cui oggetti siano rappresentazione di pizze. I nomi delle componenti devono essere: tipo, indirizzo, prezzo, consegnato. Devono essere realizzati i seguenti metodi:

  • metodi get e set per tutte le componenti; i metodi set per la componente prezzo deve verificare che il prezzo sia maggiore di zero; il metodo set per l'indirizzo deve verificare che non sia null; in caso di errore, stampare un messaggio di errore e terminare il programma;
  • metodo costruttore che ha come argomento un BufferedReader; il metodo deve leggere i valori da mettere nell'oggetto da file; questi valori sono memorizzati uno per linea; i valori booleani sono memorizzati attraverso le stringhe "si" e "no"; nel caso in cui il file finisca prima che si riescano a leggere tutte le componenti, si stampi un messaggio di errore e si esca dal programma;
  • un metodo toString che restituisce una stringa in cui ci sono tutte le componenti dell'oggetto.

Seconda parte

Definire una seconda classe Pizzeria i cui oggetti rappresentano tutte le pizze fatte dalla pizzeria in una giornata. Ogni oggetto di questa classe è quindi la rappresentazione di un insieme di oggetti Pizza Per rappresentare insiemi di oggetti, usare un array.

Costruttore della classe:

Pizzeria(String nomefile)
crea un oggetto Pizzeria che rappresenta l'insieme di tutti gli oggetti Pizza che sono memorizzati nel file il cui nome è dato dalla stringa passata come argomento; la prima linea del file contiene il numero degli oggetti memorizzati su file.

Metodi della classe:

int incassato()
ritorna il totale incassato fino a questo momento, ossia la somma del prezzo delle pizze che sono state consegnate
boolean daConsegnare(String indirizzo)
verifica se tutte le pizze da consegnare a un certo indirizzo sono state effettivamente consegnate
