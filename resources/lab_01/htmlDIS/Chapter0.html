Chapter 0 - Installing Leonardo

Hardware and software requirements

    Your system should match the following requirements to run Leonardo:

    • Macintosh Operating System version 7.5 or higher
    • PowerPC microprocessor
    • At least 3 MB of free RAM
    • At least 6 MB of free secondary memory

Leonardo's components

    Leonardo (we assume to deal with the version 3.3) is composed of:

    • an executable file (Leonardo 3.3)
    • a documentation folder (Docs)
    • a folder containing Leonardo's animation examples (Animation examples 3.3)
    • a folder containing some examples of graphs created by Leonardo's built-in graph editor (Graph Examples)
    • two folders used by Leonardo's built-in C compiler (C Headers and C Libraries)
