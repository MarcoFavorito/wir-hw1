Software Libraries for Sequential Computing

Software libraries are supposed to provide correct, efficient and easy-to-use software components of wide applicability, such as a dictionary data type, a network-flow algorithm, a discrete-event queue. Within ALCOM and ALCOM II we developed the extensive LEDA library of efficient data types and algorithms. LEDA offers a wide range of data types (lists, dictionaries, priority queues, sorted sequences, graphs, planar graphs, ...) and combinatorial algorithms (network flow, matching, planarity testing, ...). LEDA is freely available for academic use and is in fact used by several hundred academic institutions world-wide. We can build on a rich basis and have access to a well established user community.
