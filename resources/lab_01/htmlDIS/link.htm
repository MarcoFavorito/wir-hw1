Siti Web di interesse

Siti legati a Java
http://java.sun.com è il sito della Sun su Java.
http://www.jcreator.com offre software freeware e shareware di supporto alla scrittura dei programmi in Java. Di particolare interesse JCreator LE.
http://www.ericharshbarger.org/java Deposito di applet, con esempi e dimostrazioni.
http://www.eclipse.org/. Ambiente di sviluppo open source ma professionale.
Browser
http://www.opera.com. Opera, browser che supporta pienamente Java 2.
http://home.netscape.com/computing/download/index.html. Il celebre browser Netscape.
http://www.microsoft.com/windows/ie/downloads/ie6/default.asp. L'ultima versione di Internet Explorer.
http://www.mozilla.org/. Browser open source emergente, stabile ed efficiente. In particolare evidenza Mozilla Firefox, browser leggerissimo e rapido.
Software shareware o freeware
http://www.winzip.com. Celebre utilità shareware per la compressione/decompressione dei file.
http://www.borland.it/products/downloads/download_jbuilder.html. JBuilder Personal è un ambiente integrato per lo sviluppo in Java. Gratuito per gli studenti e le organizzazioni non-profit. Complesso da utilizzare. Richiede registrazione gratuita.
http://www.openoffice.org. Progetto Open Source che ha quale scopo primario lo sviluppo di una Suite per l'Ufficio multi-lingue e multi-piattaforma, basata sullo standard aperto XML di salvataggio dei documenti.
