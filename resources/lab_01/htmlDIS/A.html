Parte II

Esercizio A

Dire cosa succede quando si esegue il seguente programma. Motivare la risposta (risposte mancanti di spiegazione sono considerate nulle). 

class Mistero {
  static int prova(int x) {
    if(x==0) {
      System.out.println("Abcd");
      return 0;
    }

    System.out.println(prova(x-2));

    return x;
  }


  public static void main(String args[]) {
    System.out.println(prova(4));
    System.out.println(prova(3));
  }
}

Esercizio B

Spiegare l'algoritmo di ricerca binaria in generale ed in particolare quando va cercato l'elemento 10 nel seguente array di interi:

-3 2 4 8 11 12 49
