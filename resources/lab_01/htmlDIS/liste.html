Esercitazione 3

Gestione di dati sperimentali

Scrivere una classe Esperimento che mantenga informazioni riguardo ai dati sperimentali misurati durante un esperimento. I dati vengono forniti sotto forma di valori interi positivi memorizzati su file (uno per ciascuna linea).

La classe Esperimento deve avere le seguenti funzionalità:

  • creazione di un esperimento a partire da una stringa contenente il nome del file in cui sono memorizzati i valori
  • calcolo e restituzione della media (in double) dei valori
  • ordinamento dei valori sperimentali
  • eliminazione dei valori multipli (doppioni)
  • scrittura su nuovo file (passato come argomento) dei valori

Nota: Si usi il file dati.txt come file di dati di prova per la classe e il seguente programma Main.java per verificare il funzionamento della classe

Parte 1.

Realizzare la classe Esperimento utilizzando la tecnica degli array dinamici.

Parte 2.

Realizzare la classe Esperimento utilizzando una struttura collegata lineare (o lista) di interi. Si risolva l'esercizio implementando il metodo ordina in tre passi:
1) costruzione di un array di interi dagli elementi della lista;
2) invocazione di un metodo di ordinamento sugli array;
3) costruzione di una nuova lista contenente i dati presenti nell'array ordinato.

Per ordinare i dati, si utilizzi l'algoritmo di ordinamento quicksort, implementato nella classe QuickSortAlgorithm.java.
