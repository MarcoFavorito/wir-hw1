Springer LaTeX files for Author

Please, remember that the final version of the paper must be written in LaTeX using the format specified by Springer. This is mandatory, since otherwise the Springer will not be able to publish your paper in the Proceedings of POSTA2003. 

The name of the tex file must be: <paper ID number>.tex
All the figures must be in eps format with names:  <paper ID number>fig1.eps, ...
You can include the bibliography items into the .tex file or provide us with the .bib file

Please, check that your paper, using this format, does not exceed 8 pages including the bibliography.

Example:
for the paper 002s with two figures:
002s.tex, 002sfig1.eps, 002sfig2.eps
Any other additional file (i.e. references,...)
must have a name starting with the paper ID number

Download
You can download the LaTeX style (svmult.cls), the template text file (PostAuthor.tex) of your contribution and a sample file (PostAuthorSample.pdf).

PAPERS USING DIFFERENT STYLES OR EXCEEDING 8 PAGES WILL NOT APPEAR IN THE PROCEEDINGS
