SERVLETS

COMPILAZIONE

Aver cura di inserire nel classpath il package servlet.jar

ESECUZIONE

Inserire il codice del servlet nella directory <servlets home>\webpages\WEB-INF\servlets

 

Inserire  nel file servlets.PROPERTIES la seguent linea

<servletname>.code=<servletclass>

quindi se il servlet si chiama mioservlet e la classe corrispondente è mioservlet.class si inserirà la seguente linea

mioservlet.code=mioservlet.class

Avviare il server con il file batch startserver

il servlet si trova all'indirizzo

http://localhost:8080/servlet/mioservlet

FERMARE L'ESECUZIONE

utilizzare il file batch stopserver
