AMPL student version

L'interprete AMPL (riga di comando) con i solutori: CPLEX, SNOPT e MINOS, e' disponibile in questo unico file zip nella versione per studenti reperibile anche all'indirizzo  http://www.ampl.com/  dove e' possibile scaricare le versioni per studenti di solutori aggiuntivi come LOQO.

All'indirizzo http://www.ampl.com/GUI/expermt.html e' possibile scaricare varie GUI per l'interprete da riga di comando tra cui una interamente scritta in JAVA . Java AMPL interface source .
