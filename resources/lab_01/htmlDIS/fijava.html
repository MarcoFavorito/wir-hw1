2. Obiettivi del corso e capacita' acquisite dallo studente

Il corso di Fondamenti di Informatica I è il primo corso di informatica obbligatorio per gli studenti di Ingegneria Informatica.
Fondamenti di Informatica I  è propedeutico a tutti i corsi di informatica del corso di laurea, ed, in particolare, risulta strettamente collegato con i corsi di Principi di Progettazione del Software e Strutture Dati. L'obbiettivo del corso di Fondamenti di Informatica I  è di far acquisire allo studente alcuni principi fondamentali della programmazione ad oggetti, funzionale ed imperativa. A questo scopo viene presentato allo studente il linguaggio di programmazione JAVA. Attraverso JAVA vengono progressivamente sviluppati i concetti di: oggetto, classe, metodo, ricorsione, istruzione di controllo, struttura di un programma. Vengono quindi introdotte le strutture dati: vettori, stringhe, matrici, liste, pile, code, alberi binari. Lo sviluppo dei metodi di programmazione viene condotto con l'ausilio degli strumenti disponibili nel laboratorio didattico di Via Tiburtina. Al termine del corso lo studente sarà in grado di scrivere programmi in JAVA che richiedono la manipolazione delle strutture dati fondamentali.

3. Prerequisiti

Non si richiedono prerequisiti specifici. Si utilizzano nozioni di base di matematica e di teoria degli insiemi.

4. Programma

Parte I: Introduzione al linguaggio JAVA (4R,1L)

Introduzione al personal computing:

Il sistema di elaborazione: software di base, linguaggi macchina, software applicativo e linguaggi di programmazione ad alto livello. Strumenti di sviluppo: editor, browser, compilatori, intepreti, ambienti di programmazione

Programmazione OO in JAVA

Utilizzo di classi e creazione di oggetti e attivazione di metodi. Esempi di classi predefinite. Definizione di classe: campi e metodi. Creazione di semplici classi che estendono classi predefinite. Esempi di definizione di classi.

Altri aspetti della programmazione in JAVA

Modalità di I/O in Java, uso di file testo, Web, HTML, Gui, Applet

Programmazione imperativa in JAVA

Condizionali, cicli, Tipi di dato primitivi: numerici, caratteri, boolean. Classi e metodi per i tipi primitivi

Programmazione funzionale in JAVA

Ricorsione, modello Run Time

Parte II: Programmazione in JAVA.(5R)

Strutture dati

Definizione ed uso di array in JAVA, la classe vector, Algoritmi: ricerca, ordinamento, matrici ed array n-dimensionali
Implementazione di insiemi, liste ed altre strutture lineari tramite vector

Tecniche di programmazione

Sintassi dei linguaggi di programmazione, individuazione e correzione degli errori, cenni all'efficienza dei programmi

Altri aspetti del linguaggio JAVA

Eccezioni, Programmazione ad eventi, Threads, Ereditarieta' (cenni)

Strutture collegate lineari:

Strutture collegate lineari:  insiemi, liste, pile, code,

Strutture collegate non lineari:

Alberi binari
