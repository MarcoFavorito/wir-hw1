FONDAMENTI DI INFORMATICA II

Corso di Laurea in Ingegneria Elettronica

Università La Sapienza - Consorzio Nettuno

Appello del 26-5-2001

Parte Pratica (100 minuti, 20 punti)

Un sistema di gestione dei voli in partenza da un aereoporto gestisce i dati di sua competenza come segue:

  • per ogni volo da gestire sono memorizzat:

    • codice volo (stringa di 5 caratteri alfanumerici)

    • ora di partenza (ore e minuti soltanto)

    • data di partenza (giorno mese anno)

    • numero di posti liberi (cioè ancora occupabili al check-in)

    • destinazione (stringa di al più 30 caratteri

  • i voli sono memorizzati in una tabella realizzata mediante un array di opportune strutture

  • alle 23:00 di ogni giorno la tabella viene scaricata in un opportuno file

  • alle 6:00 di ogni giorno la tabella viene ricaricata in memoria centrale dal file in cui è memorizzata

L’esercizio richiede la progettazione di un programma per la gestione dei voli.

NB Mentre il massimo numero di voli memorizzabili in tabella può essere stabilito come una costante MAXVOLI, non è detto che in tabella ci siano sempre MAXVOLI elementi.

1)

Definire le strutture dati necessarie allo svolgimento dei punti seguenti. Specificare inoltre i prototipi delle funzioni coinvolte.

Facoltativamente scrivere una main() che usi le funzioni richieste dai punti successivi.

2)

Scrivere una funzione C che, ricevendo una tabella di voli, Tab, un codice di volo Cod e un intero k, modifichi il numero di posti disponibili del volo Cod, diminuendoli di k.

3) (facoltativo)

Scrivere una funzione C che, ricevendo una tabella di voli, Tab, un codice di volo Cod e un’ora (oo, mm), modifichi l’orario di partenza del volo Cod, sostituendogli (oo, mm).

4)

Scrivere una funzione C che, ricevendo una tabella di voli, Tab e il nome di un file, memorizzi nel file tutti i voli contenuti nella tabella.

5)

Scrivere una funzione C che, ricevendo una tabella di voli (vuota), Tab e il nome di un file contenente le informazioni su una serie di voli, riempia la tabella con tali informazioni.

6) (facoltativo)

Scrivere una funzione C che, ricevendo una tabella di voli, Tab, chieda da input i dati di un nuovo volo e lo aggiunga alla tabella.

II)
