Istruzioni per l'uso di JAVA in laboratorio


Il JDK (jdk1.17 o jdk1.2.2) installato sulle macchine del laboratorio richiede un comando preliminare che deve essere dato dalla finestra promt dei comandi (shell del DOS). Occorre digitare (nella directory di lavoro):
java11 - per la versione 1.17 o
java2 - per la versione 1.2.2

C:\TEMP>java2

C:\TEMP>

Una volta inizializzato il sistema per l'utilizzo di JDK, i file creati possono essere compilati nella directory di lavoro con il comando:

C:\TEMP>javac NOME_DEL_FILE.java

Per l'esecuzione del programma:

C:\TEMP> java NOME_DEL_FILE
