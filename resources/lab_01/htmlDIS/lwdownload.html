Download page

At the moment, the following files are avalable (version 1.0.0) :
  • The Player: a viewer for Leonardo Web presentations, available as a web applet [jar file] or as a standalone application [jar file].
  • The Builder [jar file]: a visual editor, which can be used to build and edit presentations.
  • The Library [java file]: a Java library that supports creation of batch-generated animation scripts using Java programs as drivers. Animations created in this way can be further refined using the Builder. Also an example of java code that uses the library is available.
  • Examples of Leonardo Web script [zip file]
