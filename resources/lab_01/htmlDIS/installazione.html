Installazione del Java Software Development Kit

Nel sito della Sun Microsystems troviamo le ultime versioni della Java 2 Standard Edition (J2SE) per vari sistemi operativi (Solaris, Linux e Windows). In seguito presenteremmo l'installazione per il sistema Windows (98, NT, 2000, XP).

1.    Scaricare JSDK.

Il Java Software Development Kit si può scaricare dal indirizzo http://java.sun.com/j2se . Secondo la versione Java l'indirizzo potrebbe essere cosi:

http://java.sun.con/j2se/1.4.2/download.html

Il file scaricato si chiama file j2sdk-1_4_2_<version>-windows-i586-i.exe e rappresenta il programma d'installazione del JSDK. Il file verrà memorizzato in una directory a propria scelta sul disco rigido.

Java si può installare anche direttamente dal sito, ma non consigliamo questa scelta.

2.    Installare JSDK.

La procedura d'installazione per il sistema Windows si trova all'indirizzo:

http://java.sun.com/j2se/1.4/install-windows.html

e ha i seguenti passi:

1.      Eseguite il programma d'installazione clicando due volte sul file                                  

j2sdk-1_4_2_<version>-windows-i586-i.exe .

Durante l'esecuzione seguire le indicazioni visualizzate dal programma. Alla fine il programma chiederà di riavviare il computer. Finita l'esecuzione e riavviato il computer si può cancellare il file del programma d'installazione.

Sul disco troverete una directory j2sdk1.4.2_<version> con la seguente struttura al albero:

 

                  j2sdk1.4.2_<version>
  ____________________|___________________
 |    |    |    |    |   |     |      |   |
 |    |    |    |   bin lib    |    demo  |
 |    |    | LICENSE |   |     |         jre
 |    | COPYRIGHT              |        __|__
 |  README.txt              include    |     |
readme.html                           bin   lib
 

Per avviare un programma del JSDK, ad esempio il compilatore javac.exe, si può procedere fornendo il percorso completo dei file eseguibili cosi:

C:>.\j2sdk1.4.2_<version>\bin\javac MyClass.java

2.      Modificate la variabile PATH  (Opzionale)

Se desiderate avviare i programmi java senza fornire il percorso completo dei file javac.exe o java.exe dovete aggiornare in modo persistente la variabile PATH dell'ambiente di esecuzione. La variabile PATH consiste da una serie di nomi di cartelle separati da “;”. All'avvio di un programma, Windows fa una ricerca tra tutti i programmi nelle cartelle trovate in PATH dalla sinistra verso destra. Dovete dunque avere solo una cartella per Java SDK (in caso contrario le successive saranno ignorate).

La modalità di aggiornamento della variabile PATH dipende dalla versione Windows che utilizzate.

Microsoft Windows NT, 2000 e XP

a.      Scegliete Avvio, Pannello di controllo e clicate due volte Sistema.

b.      Per Microsoft Windows NT selezionate il tab Environment; per Microsoft Windows 2000 e Windows XP selezionate il tab Avanzate e poi Variabili d'ambiente.

c.      Ricercate la variabile  "Path" tra Variabili dell'utente o Variabili di sistema.

d.      Selezionate la variabile “Path”, clicate su Modifica e aggiungete alla fine la stringa C:\j2sdk1.4.2_<version>\bin. Verificate che sia preceduta da “;”.

e.      Clicate “OK” e di nuovo “OK” o “Imposta”.

f.        Il nuovo percorso sarà considerato al prossimo avvio o alla prossima apertura della finestra “Prompt dei comandi”.

Microsoft Windows 98

a.      Con il programma Blocco Note aprite il file AUTOEXEC.BAT . Se non sapete dove trovarlo, scegliete Start, Esegui e scrivete sysedit e poi clicate OK. Tra le finestre che si aprono troverete anche una finestra AUTOEXEC.BAT. 

b.      Verificate se nel file AUTOEXEC.BAT appare la variabile PATH. Se la variabile PATH non appare, aggiungete la seguente riga nuova:

PATH=C:\J2SDK1.4.2_<version>\BIN

Se la variabile PATH già esiste nel file AUTOEXEC.BAT aggiungete alla fine della riga PATH la stringa: ;C:\J2SDK1.4.2_<version>\BIN

c.      Per fare effettiva la modifica, aprite una finestra Prompt dei comandi ed eseguite il comando:

C:> c:\autoexec.bat

Potete verificare il valore corrente della variabile PATH chiamando in una finestra Prompt dei comandi il seguente comando: 

C:> path
