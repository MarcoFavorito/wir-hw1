(COSTRUTTORI DI) TIPI DEFINITI DALL'UTENTE

consentono di scrivere speciali "ESPRESSIONI_DI_TIPO" (atte a SPECIFICARE TIPI DI DATI).

TIPO = VALORI + OPERATORI !!!

usandoli, si possono dichiarare variabili di un tipo non predefinito

var g: (Lun, Mar, Mer, Gio, Ven, Sab, Dom);

var cont: 2..7

e usarle per contenere valori previsti dal loro tipo

g:= Lun;

cont:=6;

I tipi definiti dall'utente possono essere "anonimi" come quelli sopra, o avere un nome (sempre stabilito dall'utente). Il nome viene assegnato ad un nuovo tipo mediante una DICHIARAZIONE DI TIPO, cioè un'associazione tra un identificatore (il nome di tipo) e un tipo (già esistente oppure definito dall'utente usando un costruttore di tipo).

TYPE GIORNO = (Lun, Mar, Mer, Gio, Ven, Sab, Dom);

TYPE Contatore = 2..7;

TYPE InteroNonNegativo = 0..maxint;

TYPE NumeroIntero = integer;

Se un certo nome di tipo è stato dichiarato nel programma, lo si può usare come un tipo predefinito, per definire variabili ...

var cont:InteroNonNegativo cont:= -7 cont:=5

i: NumeroIntero;

g: GIORNO g:=Gio; g:=gio g:=7

TIPO ENUMERATO (id1, ..., idn)

VALORI = {identificatori esplicitamente elencati nella definizione}

OPERATORI = ORD, PRED, SUCC

ord(Mar)=1 ord(Lun)=0 ord(Dom)=6 pred(Mar)=Lun succ(Dom)=!!!

un tipo ORDINALE (tipo ospite)

VALORI = {tutti i valori da LIM1 a LIM2 nel tipo ospite}
