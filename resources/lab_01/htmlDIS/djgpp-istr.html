Istruzioni per l'installazione di DJGPP

  1. create una directory dove installare DJGPP (es. C:\DJGPP)
  2. scaricate in questa directory tutti i file che trovate in

    3. http://ftp.dis.uniroma1.it/pub/rosati/lingtrad/djgpp
     

  3. unzippate tutti i file; potete usare sia pkunzip (con l'opzione -d) oppure il programma unzip386.exe che avete appena scaricato (con unzip386 non serve nessuna opzione)
  4. modificare il file autoexec.bat, aggiungendo le seguenti righe (se al punto 1 avete scelto una directory diversa da C:\DJGPP modificate opportunamente le istruzioni):

    5. set DJGPP=C:\DJGPP\DJGPP.ENV

    set PATH=C:\DJGPP\BIN;%PATH%

  5. a questo punto battete il comando

    6. go32-v2
     

  6. che (se tutto va bene) dovrebbe darvi come risposta qualcosa del tipo

    7. DPMI memory available: ..... Kb

    DPMI swap space available: ......Kb
     

  7. ora potete provare il compilatore C, che si invoca col comando

    8. gcc <nome-file-c>
     

  8. se riuscite a compilare, allora siete a posto. Altrimenti, se vi da' errore perche' non riesce a scrivere in una directory temporanea, dovete andare a modificare il file

    9. djgpp.env
     

  9. cambiando il valore della variabile tmpdir (in una delle primissime righe del file) in modo che punti ad una directory (temporanea) che esista e che voi possiate scrivere (es. C:\WINDOWS\TEMP)
  10. a questo punto non dovreste avere altri problemi di setup, e siete cosi' pronti per usare FLEX e BISON (che sono gia' nell'ambiente)
  11. se invece avete ancora problemi.... andate a leggervi il file readme.1st
  12. NOTA. L'installazione che avete eseguito e' solo una versione minimale di djgpp, necessaria per poter lavorare con FLEX e BISON (e il C). Chi volesse estendere tale installazione con gli altri programmi di DJGPP (es. compilatore C++), puo' andare sul sito

    13. http://sunsite.cnlab-switch.ch/ftp/mirror/simtelnet/gnu/djgpp/v2/
     
    e da li' scaricarsi anche gli altri file per completare l'installazione.
