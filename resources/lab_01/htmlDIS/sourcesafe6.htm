Visual SourceSafe 6.0d
Visual SourceSafe 6.0d contains all the service releases since version 6.0, including new minor improvements for reliability and performance. Additionally, it contains enhancements to fully integrate with Visual Studio .NET 2003. While previous 6.0 versions of Visual SourceSafe will still work with Visual Studio .NET 2003, users should upgrade in order to take full advantage of the latest source code control features of Visual Studio .NET 2003
