Configurazione ed uso del BT848

La nuova versione 0.3 del driver per la scheda frame grabber e' disponibile in ftp://ftp.dis.uniroma1.it/pub/iocchi/Linux/drivers/bt848/bt848-0.3.tgz.

Per il corretto funzionamento del driver e' necessaria la patch bigphysarea del kernel.

Istruzioni per l'installazione: 

4. Compile and install the bt848 driver

   Type 'make install' as root from the bt848 directory.
   If everithing is ok a file bt848.o will be generated,
   the devices will be created in the /dev directory,
   and the module will be loaded into the kernel.
   Check out /proc/modules to be sure it is loaded.

5. You may want to load the module during Linux bootstrap.
   Just add the following line in /etc/rc.d/rc.modules
   /sbin/modprobe bt848
