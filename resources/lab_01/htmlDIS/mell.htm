MELL - Developer Edition for MSDNAA

Description:

As a special benefit, MSDN Academic Alliance members receive three collections from the Microsoft eLearning Library (MELL), which is an effective training solution and reference for anyone learning to use Microsoft software and technologies. Each collection contains a set of Microsoft Press books and their companion content, e.g. code samples.

These three collections are included in this ISO image for MELL Developer Edition for MSDN Academic Alliance.

  • Microsoft Visual Studio .NET collection
  • Programming in Visual Basic .NET collection (NEW!)
  • General Programming collection

Here is a list of the Microsoft Press titles included in each collection.

VISUAL STUDIO .NET

  • MCAD/MCSD Self-Paced Training Kit: Developing Web Applications with Microsoft Visual Basic® .NET and Microsoft Visual C#® .NET (Exams 70-305 and 70-315)
  • Microsoft Visual C# .NET (Core Reference)
  • Programming Microsoft .NET
  • MCAD/MCSD Self-Paced Training Kit: Developing Windows-Based Applications with Microsoft Visual Basic .NET and Microsoft Visual C# .NET
  • Inside Microsoft .NET IL Assembler
  • Introducing Microsoft .NET, Second Edition
  • Inside C#, Second Edition

VISUAL BASIC .NET

  • Practical Standards for Microsoft Visual Basic .NET, Second Edition
  • Microsoft Visual J# .NET (Core Reference)
  • Coding Techniques for Microsoft Visual Basic .NET
  • OOP: Building Reusable Components with Microsoft Visual Basic .NET
  • Upgrading Microsoft Visual Basic 6.0 to Microsoft Visual Basic .NET
  • Designing Enterprise Applications with Microsoft Visual Basic .NET
  • Programming Microsoft Windows with Microsoft Visual Basic .NET

GENERAL PROGRAMMING

  • Building Solutions with Microsoft Commerce Server 2002
  • Developing International Software, Second Edition
  • Microsoft Excel 2000/Visual Basic for Applications Fundamentals
  • Software Project Survival Guide
