Parte I

Esercizio A

Scrivere un metodo (possibilmente ricorsivo) che ha come argomento una lista che contiene solo oggetti Integer e restituisce true se esistono almeno tre elementi in sequenza crescente.

Esercizio B

Scrivere un metodo che ha come argomento un albero binario e restituisce il numero dei nodi che hanno come figlio almeno un nodo interno.

Parte II

Esercizio A

Dire cosa succede quando si esegue il seguente programma. Motivare la risposta (risposte mancanti di spiegazione sono considerate nulle). 

class Chilosa {
  static int prova(int y) {
    if(y==0)
      System.out.println(y);

    System.out.println(prova(y-3));

    return y;
  }

  public static void main(String args[]) {
    System.out.println(prova(6));
    System.out.println(prova(5));
  }
}
Esercizio B

Spiegare l'algoritmo di ricerca esaustiva (o sequenziale), in generale, e in particolare come procede l'algoritmo quando va cercato l'intero 10 nel seguente array ordinato di interi:
-3 2 4 8 11 12 49
Se l'array e' ordinato (come in questo caso), in che modo trarreste vantaggio dall'ordinamento per ridurre il numero di passi effettuati dall'algoritmo?

Esercizio C

Rispondere a una sola delle seguenti domande:

  • spiegare come si realizza una interfaccia grafica in Java;
  • spiegare come si accede ad una base di dati in Java usando JDBC.
  • discutere le strutture collegate in Java: a cosa servono? Come sono fatte? Come si inserisce o si cancella un elemento da una struttura collegata?
