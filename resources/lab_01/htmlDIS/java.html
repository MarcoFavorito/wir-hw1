Progetto di traduttori in Java

Questa pagina riguarda chi intende sviluppare il progetto di un traduttore usando il linguaggio Java.

Ambiente di programmazione

Sul sito: 
    http://www.javasoft.com
e' disponibile l'ambiente di sviluppo per il linguaggio Java (JDK, Java Development Kit) (per qualunque sistema operativo).

Un ottimo corso su web per imparare Java (tutorial) e' accessibile all'indirizzo 

    http://java.sun.com/docs/books/tutorial/
 

Generatori di analizzatori lessicali

All'indirizzo: 

    http://www.cs.princeton.edu/~appel/modern/java/JLex/
trovate la home page di JLex, l'equivalente di FLEX per Java, ovvero un generatore di analizzatori lessicali scritti in Java. Da questa pagina e' possibile scaricare sia il software che la documentazione di JLex.

In alternativa a JLex, e' possibile usare JFlex, una versione (sembra) migliorata di JLex: la home page di JFlex (contenente sia il codice che la documentazione di JFlex) si trova all'indirizzo 

    http://www.informatik.tu-muenchen.de/~kleing/jflex/

Generatori di analizzatori sintattici

All'indirizzo: 

    http://www.cs.princeton.edu/~appel/modern/java/CUP/
trovate la home page di Cup, l'equivalente di BISON per Java, ovvero un generatore di analizzatori sintattici bottom-up scritti in Java.

Esiste anche JavaCC, un generatore di analizzatori sintattici top-down per Java che comprende anche la generazione di analizzatori lessicali. La home page di JavaCC si trova all'indirizzo 

    http://www.suntest.com/JavaCC
 Nota bene: JavaCC e' un software di natura diversa dai precedenti (che corrispondono alle versioni Java di FLEX e BISON), in particolare richiede un file di specifica che comprende sia la definizione del lessico e della sintassi del linguaggio da riconoscere. Inoltre, a differenza di BISON e Cup, genera un analizzatore top-down anziche' bottom-up, e' quindi in grado di trattare una classe di grammatiche diversa da quella accettata da BISON e Cup (ovvero le grammatiche LALR).
