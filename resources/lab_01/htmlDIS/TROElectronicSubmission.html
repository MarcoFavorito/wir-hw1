IEEE Transactions on Robotics 

Electronic Submission Procedures

Please submit the paper in double-column format in PDF (preferred), postscript, or zipped postscript as a single file by FTP to the site address ftp.dis.uniroma1.it/pub/ieeetro/incoming. The cover page of the paper should include paper type (Regular Paper, Short Paper, or Communication Item), paper title, authorsí names and full addresses, abstract, and keywords (index terms). The corresponding author should be clearly indicated. Please, send then a submission letter by email to ieeetro@dis.uniroma1.it including (as plain text in the message) all information contained in the cover page of the paper and the name of the uploaded file.

You may use any ftp client program to upload files (e.g. WS_FTP or Internet Explorer with Browsing Enhancements). Please note that file listing is disabled on the Transactions FTP server so you will not see the files in the incoming directory.

WS_FTP LE can be downloaded here. It is available for free to educational users, government employees (U.S. local, state, federal and military) or to non-business home users.

Browsing Enhancements is an optional component for Internet Explorer 5.x. It allows users to drag and drop files between remote ftp sites and the local computer. It is likely that this feature is already available on your computer. If it is not, please refer to the link below regarding how to install and use FTP folders.

http://support.microsoft.com/default.aspx?scid=kb;EN-US;217888

Occasionally people may have problems with FTP when their computers are behind some firewalls and proxies. The workarounds include:

1. Use another ftp client program.

2. Use a computer outside the current network.

Please report any technical problems to ieeetro@dis.uniroma1.it.


HOW TO UPLOAD A SUBMISSION BY FTP

SUBMIT A PAPER USING WS_FTP 

· Step 1. Start WS_FTP, enter ftp.dis.uniroma1.it for Host Name, enter anonymous for User ID, enter your email address for Password, and select Unix (standard) or Automatic detect for the Host Type. On the Advanced window menu, check that the "Passive transfer" box is deselected.  Click OK.

· Step 2. On the Local System window (left side), change the directory to where your file is located.

· Step 3. On the Remote Site window (right side), change the directory to /pub/ieeetro/incoming. Press enter.

· Step 4. Just below these windows, select Auto for the file format. 

· Step 5. Select the file you want to transfer, then click the transfer arrow to upload the file.

· Step 6. If permission is denied, there must be another file with the same name already in the incoming directory. Please change your filename and repeat Step 5.

· Step 7. Send a submission letter by email to ieeetro@dis.uniroma1.it including (as plain text in the message) all information contained in the cover page of the paper and the name of the uploaded file.

SUBMIT A PAPER USING INTERNET EXPLORER WITH BROWSING ENHANCEMENTS

· Step 0. If your browser version (e.g. 6.0) allows this option, check that under the Internet Option menu, in the Advanced window, the "Use passive FTP" box is deselected.

· Step 1. Enter ftp://ftp.dis.uniroma1.it/pub/ieeetro/incoming in the Address bar.

· Step 2. Open Windows Explorer, drag the file and drop it into the incoming folder. You may also copy the file in Windows Explorer and paste the file in Internet Explorer.

· Step 3. If permission is denied, there must be another file with the same name already in the incoming directory. Please change your filename and repeat Step 2.

· Step 4. Send a submission letter by email to ieeetro@dis.uniroma1.it including (as plain text in the message) all information contained in the cover page of the paper and the name of the uploaded file.

SUBMIT A PAPER USING NETSCAPE 4.X 

· Step 1. Enter ftp://ftp.dis.uniroma1.it/pub/ieeetro/incoming in the Location bar.

· Step 2. Click File and click Upload File.

· Step 3. In the File Upload dialog box, select the file you want to upload, click Open.

· Step 4. If permission is denied, there must be another file with the same name already in the incoming directory. Please change your filename and repeat Step 2.

· Step 5. Send a submission letter by email to ieeetro@dis.uniroma1.it including (as plain text in the message) all information contained in the cover page of the paper and the name of the uploaded file.

SUBMIT A PAPER USING A UNIX SYSTEM 

· Step 1. On the local system, cd the directory where your paper is located.

· Step 2. ftp ftp.dis.uniroma1.it.

· Step 3. Login in as anonymous, with your email address as the password.

· Step 4. cd /pub/ieeetro/incoming.

· Step 5. Enter the command: put <filename>, where <filename> is the name of the file you want to submit.

· Step 6. If permission is denied, there must be another file with the same name already in the Incoming directory. Please change your filename on your local system and repeat Step 5.

· Step 7. Send a submission letter by email to ieeetro@dis.uniroma1.it including (as plain text in the message) all information contained in the cover page of the paper and the name of the uploaded file.
