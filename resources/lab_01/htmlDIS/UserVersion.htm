USER VERSION

To use the application ManetChat on the middleware MOBIDIS is necessary to distribute on the PDA and then to install some programs.

Then are list the steps to install the application :
   
  1. To install the .NET Compact Framework
  2. To install the file System_SR
  3. To install the middleware MobiDIS
  4. To install the application Manet Chat

If the .Net Compact Framework and the file System_SR are been already install, you jump to the point 3.

1. .NET COMPACT FRAMEWORK

To develop and to test the application are been used two model of PDA:

Ipaq 5450 and Ipaq 5550.
   

•

For the Ipaq 5540 this is the file for install the .NET Compact Framework : netcf.core.ppc3.arm.cab
   

•

For the Ipaq 5550 this is the file for install the .NET Compact Framework : netcf.all.wce4.armv4.cab
   

•

For install the .NET Compact Framework on the other model of PDA, is necessary to see the type processor's and the operating system's present on the PDA.
   
  When to know both features above-mentioned, then you can to consult this table to see the file to download.
 
To install the Compact Framework on the PDA is necessary :
   
  1. To downloads the file on your desktop
  2. To distribute the file .CAB on you PDA
  3. To click on file ditributed on PDA
