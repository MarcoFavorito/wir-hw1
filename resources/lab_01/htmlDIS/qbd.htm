A public domain visual query system for the MS Dos platform, namely QBD*(QUERY BY DIAGRAM) is available via anonymous ftp at ftp.dis.uniroma1.it., under the directory /pub/santucci/QBD.

QBD* supports a fully visual user interaction through a graphical environment based on a diagrammatic representation of the Entity Relationship schema describing the underlying database. A novel feature of QBD* cisf a set of facilities designed to simplify the user interaction with an ER schema. Among them, the most important ones are:

  • top down browsing that allows for browsing a library of top-down refinements documenting an ER schema;
  • schema transformation that allows for converting a schema into a new version, that is "closer" to the query.

QBD* has been demonstrated at ACM Sigmod Intl. Conf. on Management of Data,1994.
