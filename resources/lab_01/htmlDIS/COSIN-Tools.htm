Library for generating

and measuring massive webgraphs

System Requirements

   
  • Operating System: Linux (with gcc compiler version 2.9)
  • Memory: 256 MB recommended
 

How to Install

  • Download and unzip the library
  • Once you've unzipped your library file, you should have a directory, called 'dis_library'
  • Change to the directory dis_library and type: 
    #  make
  • In the bin directory, there'll be the binary files of all procedures.
 

Note:
  • The utilities, named utl_* , are specifically designed to be used by other procedures and they are not for stand-alon runs.
  • For detailed information and examples, see the documentation.
