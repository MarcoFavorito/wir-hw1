GSview per Windows 95/98 e Windows NT

GSview e' un'interfaccia grafica per Ghostscript. Ghostscript e' un' interprete per il linguaggio PostScript, ovvero il linguaggio di definizione di stampa usato per le stampanti laser. GSview offre la possibilita' di selezionare le pagine di un documento PostScript (estensione ".ps") per la consultazione a video o per la stampa.
 

Download e installazione

Per installare GSview e Ghostscript sul proprio sistema Windows a 32 bit (Windows 95/98 o Windows NT), è necessario effettuare il download dei seguenti files:
  • gs550fn1.zip
  • gs550fn2.zip
  • gs550ini.zip
  • gsv26550.exe

Disporre i file scaricati in una directory, ed eseguire il programma gsv26550.exe, che installera' automaticamente l'applicazione.

Sul sito ftp (http://ftp.cs.wisc.edu) e' possibile trovare versioni di GSview per altri sistemi operativi.
