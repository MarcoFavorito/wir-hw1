ALCOM-IT Project

Project workplan

The workplan of the ALCOM-IT project is divided into the following areas:

  • Work Area 1 - Software Libraries for Sequential Computing.
  • Work Area 2 - Software Libraries for Parallel and Distributed Computing.
  • Work Area 3 - Foundations and Methods.
  • Work Area 4 - Software Development for specific IT demands.
  • Work Area 5 - Coordination.
