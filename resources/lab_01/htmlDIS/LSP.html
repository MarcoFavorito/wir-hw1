Introduction

The component provides functions for computing all pairs shortest paths in a weighted directed graph with nonnegative integer edge weights. The current implementation of component LSP is based on the Uniform Paths Algorithm [Camil Demetrescu, Stefano Emiliozzi, and Giuseppe F. Italiano, Experimental Evaluation of Dynamic All Pairs Shortest Path Algorithms, 2003].
