Reusable Modular Mobility Behavior Systems

Download the presentation (PPT, PowerPoint)

This lecture will focus on reusability and rapid system integration. First, we will show how we can build behavior systems by reusing existing components/modules. Then we discuss the prerequisits which enable an easy integration, in particular, the properties of reusable modules and the properties of an integration platform. Examples of successful system integration will be given, including the integration of sensors, sensor filters, and behavior code.
