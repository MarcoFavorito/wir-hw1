Abstract

The purpose of this thesis is to find an efficent solution to the planning problem in
a very dynamic ambient.
The problems we have to deal with are:
  • Noisy data
  • Unpredictable environment
  • High speed motion
The basic idea is to create a software module that calculate the best trajectory for the robot,
at every control cycle (that is about every 100 milliseconds) knowing:
  • The robot initial position
  • The final desired destination
  • The obstacle configuration
Once the trajectory is calculated another software module will produce the necessary
control command to make the robot follow the given trajectory.
Wath we expect from the overall navigation method is:
  • Low computational cost
  • High speed execution
  • Good performance also in unpredictable situation
The method we followed is based on an algorithm already used by  Konolige    ( A Gradient Method for Realtime Robot Control ).
The basic idea for the method is to calculate a navigation function, building a numeric potential field,
and find the right trajectory, following the gradient of the function.
The experiment have been made in the Robocup environment.
