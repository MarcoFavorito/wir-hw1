The Dipartimento di Informatica e Sistemistica of the University of Rome "La Sapienza" is involved (see Detailed Description of the RTD project for further information) in the following areas of the ALCOM-IT project:

  • LEDA (Work Package 1.1).
  • Automatic Graph Drawing (Work Package 1.2).
  • A Framework for Branch-and-Cut Algorithms (Work Package 1.3).
  • Foundations and Methods (Work Package 3).
  • Graph Drawing in Software Systems ( Work Package 4.4).
  • Information Dissemination and Take-Up of Results (Work Package 5.1).
  • Project Management (Work Package 5.2 ).
