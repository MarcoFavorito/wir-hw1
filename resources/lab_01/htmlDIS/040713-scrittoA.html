Durata: 1 ora e 45 minuti

Parte I

  1. Una sequenza di N numeri si dice bitonica se i primi K numeri (dove 0<=K<=N) sono ordinati in modo crescente e i restanti N-K numeri sono ordinati in modo decrescente (ad esempio, 2, 4, 5, 7, 3, 1 è una sequenza bitonica). Scrivere un metodo statico sequenzaBitonica (possibilmente ricorsivo) che, data una LinkedList (o un array) di Integer, restituisce TRUE se rappresenta una sequenza bitonica, e FALSE altrimenti.
  2. Scrivere un metodo statico livelloFogliaSinistra che, dato albero binario, restituisce il livello della foglia più a sinistra nell'albero (si assuma che la radice ha livello 0, i figli della radice hanno livello 1, ecc.), oppure -1 se l'albero è vuoto.

Parte II

Domanda 1

Dire cosa stampa il seguente programma:

public class Mistero {
    public static int mistero(int x){
        if (x==0) return 0;
        else if (x<0) return 1+mistero(x+1);
        else return 1+mistero(x-1);
    }
    public static void main(String[] args){
        System.out.println(mistero(-10));
        System.out.println(mistero(10));
    }
}

Quale funzione matematica f(x) è calcolata dal metodo mistero? Motivare le risposte (N.B. risposte non motivate saranno considerate nulle).

Domanda 2

Illustrare l'algoritmo di ordinamento a bolle (Bubblesort) usando come esempio il seguente array di interi:

Valutare il tempo richiesto dall'algoritmo in funzione del numero di elementi n dell'array nel caso peggiore e nel caso migliore usando la notazione asintotica. Motivare la risposta (N.B. risposte non motivate saranno considerate nulle).
